#  SQL Chat Agent

An AI-powered database assistant that allows users to interact with SQL databases using natural language.  
Built using LangChain Agents, Groq LLM, Streamlit, and SQLAlchemy.

Instead of writing SQL queries manually, users can simply ask questions and the AI agent generates and executes SQL queries to retrieve answers.


## 🚀 Features

-  Chat with your SQL database using natural language
-  LLM-powered query understanding
-  - 🔎 Automatic SQL query generation
- 📊 Retrieve database insights instantly
